# Logistic-regression
Project Description: Credit Risk Logistic Regression

This project aims to analyze and predict credit risk using a Logistic Regression model. The objective is to classify customers into two categories: those who are likely to default on a loan and those who are likely to repay it.

The dataset includes customer-related information such as income, age, loan details, credit history, and other financial attributes. The data is first preprocessed by handling missing values, encoding categorical variables, and selecting relevant features. Exploratory Data Analysis (EDA) is performed to understand patterns and relationships between variables.

A Logistic Regression model is then built and trained on the dataset to estimate the probability of default. The model is evaluated using performance metrics like accuracy, confusion matrix, precision, recall, and ROC-AUC score.

Overall, this project demonstrates how machine learning can be applied in the finance sector to assess risk, improve decision-making, and minimize loan defaults.
